AristoRutas
es una aplicación web de gestión de rutas de transporte construida con Django y PostgreSQL, Permite a usuarios consultar rutas 
(urbanas, interurbanas, interdepartamentales), buscar destinos y acceder a reportes según su plan de membresía. Cuenta con sistema de autenticación, 
pagos integrados y control de acceso por suscripción. Desplegada en Railway.

enlace a la pagina: https://aristorutas-production.up.railway.app/
